# Pasanaku MVP

Aplicación base para la gestión de Pasanaku (sistema de ahorro comunitario). Permite visualizar el estado financiero de un grupo, calculando los pozos y fondos comunes de forma dinámica.

## 🚀 Arquitectura del Proyecto

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
